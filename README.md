소스코드를 받은후 주석에 따라 수정하세요.
mealbot.py 와 parser.py 파일을 수정하시면 됩니다.
모두 수정하셨으면 실행법은
```
cd (소스가 들어있는 디랙터리)
python3 mealbot.py
```
입니다.(리눅스 기준)
